Use the following steps to inspect the GlyQ-IQ search results for a Thermo .Raw file

1) Run GlyQIQViewer.exe
2) Click "View and Annotate"
3) Drag the following 4 files onto the Setup pane

  - Raw file (.raw)
    - It's possible that the peaks file needs to be made ahead of time (not sure)
  - Parameter file for the targeted system (.xml)
    - For example: BasicTargetedWorkflowParameters2.xml
  - A file for the dataset specific parameters (.rtf)
    - For example: GlyQ-IQ-ViewerParameters.rtf
  - The GlyQ-IQ Results file (.txt)
    - For example: ExampleResults.txt

4) The data should load and annotated spectra should be visible
